This folder contains examples for different models.

*mnwk: Cell phylogeny information in newick format.
*mtx: The mtDNA mutation information in sparse matrix format.
*muts.csv: The name of simulated mtDNA mutations in the mtx file.
*cells.csv: The name of simulated cells in the mtx file.
*lookup_cells.csv: A table containing the mapping information between original cell names and modifed cell names in tree files.
